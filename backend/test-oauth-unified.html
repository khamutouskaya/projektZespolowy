?? Setup Instructions

Google OAuth:

  1. Go to Google Cloud Console
  2. Create project ? Enable Google Sign-In API
  3. Create OAuth 2.0 Client ID (Web application)
  4. Add to Authorized JavaScript origins:
    • http://localhost:5076 (dla API)
    • http://localhost:8000 (dla HTML - Python server)
  5. Copy Client ID and paste above
  6. Add to appsettings.json: "OAuth:Google:ClientId"

Facebook OAuth:

  1. Go to Facebook Developers
  2. Create App ? Choose "Consumer" type
  3. Add Facebook Login product
  4. Settings ? Valid OAuth Redirect URIs: http://localhost:5076
  5. Copy App ID and paste above
  6. Add to appsettings.json: "OAuth:Facebook:AppId"
